Arquitetura de Dados

Hybrid Cloud Core

Frontend PWA

Single Page Application (SPA) construída com Vanilla JS Module Pattern para performance máxima. Zero frameworks pesados (React/Angular) para load time < 500ms. Build tool: Vite (Rollup) c/ target ESNext.

Realtime (Hot)

Firebase Firestore para dados quentes: Chat, Notificações, Status de Tarefas e Presença Online. Sincronização em milissegundos via WebSockets nativos do SDK v9.

Analytics (Cold)

Supabase (PostgreSQL) para dados estruturados, relatórios complexos e histórico de auditoria (Logs). Permite queries SQL profundas para Business Intelligence.

INTEGRAÇÃO AI: O sistema consome APIs do Google Gemini Pro via Edge Functions para análise contextual de tarefas e geração de conteúdo.
