Recalculação Retroativa de Pontos

Recalcula e aplica pontos retroativos de todos os módulos para todos os usuários

Atenção

Este script não exclui nenhum dado. Ele recalcula os pontos a partir das ações reais dos usuários (tarefas aprovadas, depoimentos, fórum, etc.) e atualiza o saldo total. Se o usuário já possui pontos contabilizados, a diferença será ajustada para cima.

📋 Kanban ❤️ Depoimentos 💬 Fórum 📅 Rotina ADM 📝 Relatórios 📚 Cursos ✅ Aplicar
